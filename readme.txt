Title: Forest Horizon

Project Description: My TP is an aerial view survival game. The player will be able to choose from two classes, each with unique abilities. They will then be sent into the middle of a forest and have to fight against waves of birds that approach from the edges of the screen. There are different types of birds that start and stop appearing at certain times and the enemy spawn rate also increases progressively over the course of the game.

How to run: Simply open the game.py file and run the code.

Libraries needed: pygame, (also math, random, and copy)

Shortcut commands:
Press the ESC key to automatically go back to the title screen.
When selecting a character, if C is being held, game will enter test mode where enemies don't spawn by themselves. The 1 2 3 4 keys can be used to spawn different enemies at the cursor's location.
